# Minha CVC - Automação

Projeto do App Minha CVC

  - Fazer um clone do projeto
  - Executar o comando ``` bundle install``` (Instala todas dependencias para executar o projeto)
  - Executar o comando ``` cucumber features/login.feature -p (perfil) -t (@tag) ``` para iniciar os testes.
  Exemplo: cucumber features/login.feature -p android -t @Login

# Relatorio
A automação gera relatório ```(data/report/minhacvc_report.html)```com os dados da execução, destino utilizado, tempo de execução e evidência dos testes
